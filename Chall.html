Security

IoT presents a peculiar security challenge because of the massive number of connected devices that expand the attack vector and secondly because of the computation and memory constraint that makes it impossible to install an elaborate security software in the devices. To overcome this challenge innovative security approaches are being developed and deployed. Some of the IoT security tools in use are Zero Trust, hardware security, etc.

Privacy

Massive data is collected and processed with IoT. These data is either personal or societal/national. Thus how personal data is used raises important privacy questions. So countries have introduced data protection rules to protect their citizens. The GDPR inacted by EU is a case in point. However, this issue is controversial and still deliberation on how to come up with the best privacy protection without stifling innovation is an open question.

Safety

It is important to make sure that software operation in IoT is error free espeially in devices that can put life in hazard. Medical devices, autonomous vehicles, and such devices closely related to human life should be safe or fault proof. The safety of mission critical applications is also important even if it does not involve hazard to humans due to the huge loss they incur
